# "Змейка"

## О чем этот проект?

Игра где змейка двигается за яблоком

### Как играть?

- Змейка двигается сама, вы только выбираете направление
- Нужно подводить голову змейки к яблоку
- Нельзя врезаться в свое тело
- Если змейка уходит за край экрана, она появляется с другой стороны

## Управление

Это очень просто:
- **Стрелка вверх** ↑ — двигаемся вверх
- **Стрелка вниз** ↓ — двигаемся вниз  
- **Стрелка влево** ← — двигаемся влево
- **Стрелка вправо** → — двигаемся вправо
